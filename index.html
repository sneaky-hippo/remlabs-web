Neuroscience Research

How dreaming makes
agents smarter.

The Dream Engine is grounded in sleep science. Every stage of the 9-stage pipeline maps to a biological process that occurs in the mammalian brain during memory consolidation. We didn't guess — we replicated.

🧠
Hippocampal Consolidation
Memories transfer overnight.
During NREM sleep, the hippocampus replays recent experiences through sharp-wave ripples, transferring memories to the neocortex for long-term storage. Dream Engine's synthesize stage mirrors this: raw inputs are replayed, structured, and written to persistent indexed storage.
⚡
REM Free Association
Cross-domain insight emerges.
REM sleep is characterized by an acetylcholine surge that suppresses hippocampal gating, allowing distant memories to associate freely. The Dream Engine's associate and insight_generate stages replicate this — surfacing non-obvious connections across your agent's entire knowledge graph.
🎯
Targeted Memory Reactivation
Critical memories get priority.
TMR is a proven neuroscience technique where sensory cues during sleep enhance consolidation of specific memories. Our TMR system lets you tag high-priority memories so the Dream Engine processes them deeply — not uniformly. You control what compounds fastest.
9-Stage Pipeline
From raw memory to refined intelligence.
Each stage maps to a specific phase of biological sleep. Together, they run every night.
01 synthesize Consolidates fragmented memories. Mirrors NREM Stage 2 sleep-spindle replay — raw experience becomes structured knowledge.
02 pattern_extract Identifies recurring structures. Analogous to thalamocortical oscillations during deep sleep — the brain finding signal in noise.
03 insight_generate Surfaces non-obvious connections. Replicates REM-phase acetylcholine-driven free association — where creative leaps emerge.
04 compress Removes redundancy. Mirrors slow-wave sleep's synaptic downscaling — preserving signal, pruning noise, reducing storage by up to 40%.
05 associate Cross-links disparate memories. Based on hippocampal indexing theory — building semantic graph edges between memories that appear unrelated.
06 validate Scores confidence and resolves contradictions. Analogous to prefrontal cortex error-correction during late-cycle sleep.
07 evolve Adapts memory structure to recent patterns. Mirrors experience-dependent plasticity — synapses that fire together, wire together.
08 forecast Generates temporal predictions from extracted patterns. Based on research into sleep's role in prospective memory and anticipatory planning.
09 reflect Meta-cognitive review of the entire cycle. Inspired by post-sleep consolidation — the brain's morning assessment of what it actually learned.

Developer API

Built on a real API.

REM Labs is the consumer layer. Underneath is slopshop.gg — full REST API, CLI, MCP server, 1,303 endpoints.

# Store a memory curl -X POST https://api.slopshop.gg/v1/memory/store \ -H "Authorization: Bearer $SLOP_KEY" \ -d '{"content":"User prefers concise responses. Expert in Rust."}' # → { "id": "mem_9a3z1", "stored": true } # Start a Dream cycle curl -X POST https://api.slopshop.gg/v1/memory/dream/start \ -H "Authorization: Bearer $SLOP_KEY" \ -d '{"strategy":"synthesize,compress,insight_generate"}' # → { "dream_id": "drm_7f2b", "status": "running", "stages": 9 }
# Install npm install -g @slopshop/cli # Store from stdin echo "User is a senior engineer, prefers dark mode" | slop memory store # → ✓ Stored mem_k9x2 # Run a full Dream cycle slop dream run --all-stages # → Intelligence Score: 87.4 | Insights: 12
# Add to Claude Code or any MCP client (~/.claude/mcp.json) { "mcpServers": { "slopshop": { "command": "npx", "args": ["-y", "@slopshop/mcp-server"], "env": { "SLOP_KEY": "your-key" } } } } # Claude now has 1,303 tools. Memories persist across sessions.
